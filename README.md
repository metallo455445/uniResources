# Scopo 

Repositorio nel quale archivio lavori dell'università tra relazioni e progetti extra. 

# Come funziona

## Scegliere la branch

La branch "stabile" è quella del mese fissato in default, la versione più aggiornata è del mese corrente

## Cartelle

Sono presenti le cartelle delle varie esperienze di laboratorio, in ogniuna di queste cartelle c'è il codice python per l'analisi dati e un backup della relazione. <br>

Nella cartella funCodes ci sono dei codici simpatici che ogni tanto faccio per sfizio. <br>

la cartella defaultDocuments contiene degli "scheletri" di codici python e latex. Ogni volta che devo scrivere un documento nuovo lo copio da qui. <br>
